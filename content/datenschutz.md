### XMPP Service
Generell werden nur die zum Betrieb des Servers notwendigen Daten auf dem Server gespeichert. Im Detail werden folgende Daten gespeichert:

a) Bestandsdaten - Informationen, die bei der Registrierung entstehen
  - JID (XMPP-ID), bestehend aus Benutzername und Domain
  - Passwort (als SCRAM-SHA1 Hash gespeichert)
  - Datum und Uhrzeit, der Nutzer Registrierung

b) Nutzungsdaten - Informationen, die bei der Verwendung des XMPP-Dienstes entstehen
  - Roster ( Kontaktliste, falls vorhanden )
  - Online-Status und Status-Nachrichten
  - Offline Nachrichten
  - Nachrichtenarchiv ( XEP-0313 - MAM )
  - HTTP Uploads / Dateiuploads
  - VCard ( Nutzer Profil)
  - Lesezeichen / Bookmarks für gespeicherte Multi-User Chats ( MUCs )

c) Verkehrsdaten - Informationen, die durch die Nutzung des XMPP-Dienstes anfallen
  - Datum und Uhrzeit, der Letzten Nutzer Aktivität. ( Login bzw. Logout )

Im Falle der Speicherung von Daten in Logfiles ist dies nach spätestens sieben Tagen der Fall. Eine darüberhinausgehende Speicherung ist möglich. In diesem Fall werden die IP-Adressen der Nutzer gelöscht oder verfremdet, sodass eine Zuordnung des aufrufenden Clients nicht mehr möglich ist.
